04

Hàm tiện ích

Các hàm static dùng kèm với hằng số để tạo giá trị nhanh.

Colors.hex()
Color hex(String hex)
Chuyển chuỗi hex #RRGGBB thành đối tượng Color. Dùng khi cần màu tuỳ chỉnh ngoài bảng.
Colors.hex("#FF6B6B")
Colors.opacity()
Color opacity(Color c, float o)
Tạo màu với độ trong suốt từ 0.0 (trong suốt) đến 1.0 (đục hoàn toàn). Dùng cho nền mờ, overlay.
Colors.opacity(Colors.PRIMARY, 0.3f)
FontStyle.font()
Font font(int size, int style)
Tạo đối tượng Font với font Arial, cỡ và kiểu chỉ định. Kết hợp hằng số size + style.
FontStyle.font(FontStyle.LG, FontStyle.BOLD)
